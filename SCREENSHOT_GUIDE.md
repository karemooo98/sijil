# Screenshot Guide (سجّل | Sijil)

This guide helps you capture App Store screenshots for **سجّل | Sijil**.

## iOS (Simulator)

1. Run the app on an iPhone simulator (e.g., 6.5" display).
2. In the Simulator menu: **File → Save Screen Shot** (or `Cmd + S`).

## Android (Emulator)

1. Run the app on an Android emulator.
2. Click the **camera** icon in the emulator toolbar to capture a screenshot.

## Tips

- Keep screenshots clean (no debug banner, no test data if possible).
- Capture key flows: login, dashboard, attendance, profile, documents.
