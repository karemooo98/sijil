## Quick steps

### iOS (Simulator)
- Run the app on an iPhone simulator.
- Simulator menu → **File → Save Screen Shot** (or `Cmd + S`).

### Android (Emulator)
- Run the app on an Android emulator.
- Click the **camera** icon in the emulator toolbar.
